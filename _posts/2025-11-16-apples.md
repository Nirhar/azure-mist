An apple is a sweet, edible fruit produced by an apple tree.

Apple trees are cultivated worldwide, and are the most widely grown
species in the genus *Malus*. The tree originated in Central Asia, where
its wild ancestor, [*Malus sieversii*](https://en.wikipedia.org/wiki/Malus_sieversii),
is still found today. Apples have been grown for thousands of years in Asia and Europe,
and were brought to North America by European colonists.

## Varieties

There are over 7,500 known cultivars of apples. Some of the most popular include:

- **Fuji** — sweet, crisp, originally bred in Japan
- **Granny Smith** — tart, firm, great for baking
- **Honeycrisp** — juicy, balanced sweet-tart flavour

See the full list on [Wikipedia's apple cultivar page](https://en.wikipedia.org/wiki/List_of_apple_cultivars).

## Classifying Apples in Code

Here's a simple Python dataclass to represent an apple:

```python
from dataclasses import dataclass

@dataclass
class Apple:
    variety: str
    colour: str
    sweetness: int  # 1 (tart) to 10 (very sweet)

    def is_good_for_baking(self) -> bool:
        return self.sweetness < 5

granny_smith = Apple(variety="Granny Smith", colour="green", sweetness=3)
print(granny_smith.is_good_for_baking())  # True
```

And the same idea in JavaScript:

```js
const apple = {
  variety: "Fuji",
  colour: "red",
  sweetness: 8,
  isGoodForBaking() {
    return this.sweetness < 5;
  }
};

console.log(apple.isGoodForBaking()); // false
```
